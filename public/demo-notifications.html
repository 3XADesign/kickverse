Sistema de Notificaciones

Sistema profesional de modales para reemplazar los alerts nativos del navegador. Haz clic en los botones para probar cada tipo de modal.

Ejemplos de Uso

// Modal de éxito
showSuccessModal('Producto añadido al carrito', () => {
    console.log('Modal cerrado');
});

// Modal de error
showErrorModal('Error al procesar el pedido');

// Modal de advertencia
showWarningModal('Completa todos los campos');

// Modal de confirmación
showConfirmModal('¿Eliminar este elemento?',
    () => console.log('Confirmado'),
    () => console.log('Cancelado')
);
